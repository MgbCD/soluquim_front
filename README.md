# Soluquim Front

Este repositorio contiene el código del **frontend** del módulo de pedidos de Soluquim, desarrollado con **React** y **Material UI**.

El objetivo del proyecto es gestionar los pedidos de elementos de protección personal (EPP), permitiendo:
- Crear nuevos pedidos asociados a una empresa.
- Listar todos los pedidos existentes con paginación.
- Ver detalle de cada pedido con información de empresa, estado y valor total.


## Tecnologías utilizadas

- **React 18**
- **Material UI (MUI)** para la interfaz visual moderna y responsiva.
- **Axios** para la comunicación con el backend.
- **React Router DOM** para la navegación entre vistas.
- **Context API** para el manejo global del estado.
